# Real-Time Routes Tracker

A real-time device tracking web application built using Node.js, Express, Socket.io, and Leaflet.

## Features
- Real-time location tracking
- Multi-user support
- Interactive map (Leaflet)
- Live updates using WebSockets

## Tech Stack
- Node.js
- Express.js
- Socket.io
- Leaflet.js
- HTML, CSS, JavaScript
